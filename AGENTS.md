# AGENTS.md

This repository is a premium personal portfolio website.

## Goals

Build a portfolio that impresses employers and recruiters with:

- premium design
- strong visual hierarchy
- polished motion
- responsive implementation
- clean code quality

## Required Stack

- React
- TypeScript
- Next.js unless explicitly told otherwise
- Tailwind CSS for styling by default
- Motion for React animations by default

## TypeScript

- Use TypeScript for all new files.
- Prefer strict typing.
- Avoid `any` unless there is no better option.

## Styling

- Prefer Tailwind CSS.
- Do not add Sass/SCSS unless explicitly requested.
- Use CSS Modules or plain CSS only for cases Tailwind does not handle cleanly.

## Animation

- Prefer Motion for most UI animation work.
- Use GSAP only when complex timeline-based or advanced scroll-driven animation is clearly needed.
- Keep motion tasteful and premium.
- Avoid noisy or excessive animation.

## Design

- Avoid generic template-like SaaS design.
- Prioritize typography, spacing, hierarchy, and polish.
- Keep the visual system cohesive.
- Focus on making the portfolio feel custom and memorable.

## Quality

- Fully responsive.
- Accessible and semantic.
- Keep dependencies minimal.
- Optimize performance.
- Reuse components where it helps maintainability.

## Workflow

- For larger tasks, plan before implementation.
- Implement in slices/sections.
- Verify visual polish, responsiveness, and animation quality after major changes.
